Cloudflare Pages via Terraform

Admin console keeps the component in the app and the Pages project in infra.

This example is geared toward Git-backed Pages projects where Gluon should reconcile the Cloudflare contract but keep the app manifest beside the code.

What this component shows

  • pnpm workspace installs and Turbo builds from the repo root
  • Terraform-managed Cloudflare Pages projects
  • App-local component manifests that still point at infra code
